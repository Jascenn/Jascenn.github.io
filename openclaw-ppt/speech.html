OpenClaw 入门演讲稿

《给自己配一个真正的 AI 助手》

演讲者：凌一 | 时长参考：约 20-30 分钟



第1页 · 封面


大家好，我是凌一。


今天我想跟大家聊一件事——我们每天都在用 AI，但你有没有觉得，这些 AI 工具用起来总差点意思？


今天我要介绍的这个东西，叫 OpenClaw。它让我第一次觉得，AI 真的是"我的"了。



第2页 · 你有没有这些烦恼？


我先问大家几个问题：


- 你有没有跟 ChatGPT 聊了半天，第二天打开，它完全不记得你说过什么？

- 你有没有想在手机上用 AI，但只能打开网页，很麻烦？

- 你有没有担心过，自己跟 AI 说的那些私密的事，都存在别人的服务器上？


（停顿，让观众点头）


对，这就是现在大多数 AI 工具的现状。



第3页 · 现有 AI 工具的3个问题


总结一下，现有工具有三个核心问题：


第一，没有记忆。 每次对话都是从零开始，你得反复介绍自己，反复解释背景。


第二，渠道单一。 只能在特定的网页或 App 里用，不够灵活。


第三，隐私风险。 你的数据全在别人的服务器上，你没有控制权。


这三个问题，OpenClaw 全部解决了。



第4页 · OpenClaw 是什么？


OpenClaw 是什么？


我喜欢用一个类比来解释：


你想象一下，现在的 AI 就像一个很聪明的人，但他每天都失忆，只能在一个固定的房间里见你，而且他住在别人家。


OpenClaw 做的事情，就是给这个 AI——


- 装上一个大脑，让它记住你

- 给它一个身体，让它能在各种地方找到你

- 给它一个家，就在你自己的设备上


简单说：OpenClaw 是一个开源框架，让你在自己的设备上部署、管理、扩展 AI 助手。



第5页 · 它能做什么？


说几个真实场景：


- 早上起来，在 Telegram 问它"今天天气怎么样，要带伞吗？"

- 下午开会前，它主动提醒你"你有个会议还有20分钟"

- 晚上你说"帮我把今天的工作总结一下"，它直接给你写好

- 你在飞书群里 @ 它，它帮你查资料、整理信息

- 它记得你上周说要写一篇文章，今天主动问你"写了吗？"


这些，都是真实可以做到的。



第6页 · 多渠道接入


OpenClaw 支持的渠道非常多：Telegram、微信、飞书、Discord、iMessage、Slack……


你在哪，它就在哪。


就像你有一个朋友，不管你发微信还是发短信，他都能收到，都能回你。



第7页 · 记忆系统


这是我最喜欢的功能。


OpenClaw 有一套持久化记忆系统。它会记住你的名字、你的习惯、你说过的事、你的偏好。


重启也不会忘。


我自己用了一段时间之后，它已经知道我喜欢简洁的回答、知道我在做自媒体、知道我的工作节奏。


这种感觉，才叫"我的 AI 助手"。



第8页 · 技能系统


OpenClaw 有一个技能系统，就像手机的 App Store。


你可以给它安装各种技能：天气查询、文档助手、视频摘要、代码助手……


安装方式也很简单，一行命令：

clawhub install 技能名


社区里已经有很多现成的技能，你也可以自己写。



第9页 · 多 Agent


更厉害的是，OpenClaw 支持多个 AI 助手同时运行。


你可以设置一个"个人助手"处理日常事务，一个"工作助手"专门处理工作，一个"研究助手"帮你查资料……


不同任务交给不同的专家，效率翻倍。



第10页 · 隐私安全


很多人问我：数据安全吗？


OpenClaw 是本地部署的。你的数据，在你自己的电脑或服务器上，不经过任何第三方。


你说的每一句话，都只有你自己知道。



第11页 · 安装有多简单？


好，说了这么多，安装难不难？


三步，5分钟：


# 第一步：安装

npm install -g openclaw




第二步：初始化


openclaw onboard --install-daemon




第三步：启动


openclaw gateway start


就这样，你的 AI 助手就跑起来了。


（停顿）


不需要会编程，不需要懂服务器，跟着文档走就行。

Q：会不会很难维护？
