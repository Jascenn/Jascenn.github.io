Notion AI Agent 演讲稿

《让 Notion 学会自己思考》

演讲者：凌一 | 时长参考：约 20-30 分钟



第1页 · 封面


大家好，我是凌一。


上一场我们聊了 Notion Worker——让 Notion 自动执行任务。


今天我们升级一下，聊聊 Notion AI Agent——让 Notion 学会自己思考。


这是一个更有意思的话题。



第2页 · Worker 和 Agent 有什么区别？


先说清楚两者的区别，很多人搞混。


Worker 是执行者。你告诉它"每周五生成周报"，它就每周五生成。规则固定，不会变通。


Agent 是决策者。你告诉它"帮我管好这个项目"，它会自己分析情况，自己决定做什么，自己调整策略。


一个是按指令做事，一个是自主判断。



第3页 · 什么是 Notion AI Agent？


Notion AI Agent，我用一个类比：


你有一个助理，这个助理不只是帮你执行任务，还会主动读你的笔记，理解你的工作状态，主动提出建议，甚至在你没想到的时候提醒你该做什么。


这就是 Agent 和 Worker 最大的区别——主动性。



第4页 · Agent 能做什么？


Notion AI Agent 有五个核心能力：


- 🧠 分析任务优先级 — 自动判断什么最重要

- 📋 自动生成项目计划 — 输入目标，自动拆解步骤

- 💡 主动提出建议 — 读懂你的笔记，给出行动建议

- 🔍 跨数据库自主决策 — 综合多个数据库做判断

- 📝 读懂你的意图 — 不只是执行，而是理解



第5页 · 真实场景1：AI 分析任务优先级


你的任务列表里有20件事，你不知道先做哪个。


Agent 会自动分析：截止日期、重要程度、任务之间的依赖关系，然后给你排出一个优先级列表。


不是随机排，是真正基于逻辑的判断。



第6页 · 真实场景2：自动生成项目计划


你在 Notion 里写了一句话："做一个 AI 工具的产品介绍页"。


Agent 读到这句话，自动帮你拆解成：

- 竞品调研（3天）

- 内容框架（1天）

- 文案撰写（2天）

- 设计配图（2天）

- 发布上线（1天）


并且自动填入你的项目数据库，设好截止日期。



第7页 · 真实场景3：AI 读懂笔记主动建议


你最近在 Notion 里记了很多关于 AI 工具的笔记，但都是零散的。


Agent 读完这些笔记，主动跟你说：


"你最近研究了很多 AI 工具，要不要整理成一篇文章？我帮你列个大纲。"


这种主动性，才是 Agent 最有价值的地方。



第8页 · 真实场景4：跨数据库自主决策


你有三个数据库：项目库、人员库、资源库。


来了一个新任务，Agent 同时查看三个库，自动判断：


"这个任务适合张三做，因为他这周任务最少，而且有相关经验。"


然后自动分配，自动通知张三。


这就是跨库自主决策。



第9页 · Agent 的4个核心组件


要理解 Agent，需要知道它的四个核心组件：


- 🧠 记忆 — 长期记忆（你的偏好、历史）+ 短期记忆（当前任务上下文）

- 🔧 工具 — 能调用的 API、数据库、外部服务

- 📋 规划 — 把大目标拆解成小步骤

- ⚡ 执行 — 调用工具，完成每个步骤


这四个组件组合在一起，就是一个完整的 Agent。



第10页 · 怎么搭建？


搭建 Notion AI Agent，需要三层：


底层：Notion 数据库 — 存储你的数据和上下文


中层：LLM（大语言模型） — Claude 或 GPT，负责理解和推理


上层：Agent 框架 — 负责协调记忆、工具、规划、执行


三层搭起来，Agent 就跑起来了。



第11页 · 推荐方案怎么选？


三个主流方案：


- OpenClaw — 开源、自托管、我最推荐，和 Notion 集成很好

- LangChain — 最灵活，但需要写代码，适合开发者

- Dify — 可视化界面，适合不想写代码的人


如果你是小白，从 Dify 开始；如果你想要完全控制，用 OpenClaw。



第12页 · 避坑指南


Agent 有三件事最容易出问题：


第一，给 Agent 设边界。 别让它有权限修改重要数据，先给只读权限，测试稳定了再开放写权限。


第二，先小范围测试。 在一个测试数据库里跑，确认逻辑正确再上正式环境。


第三，保留人工审核环节。 重要决策，让 Agent 给出建议，但最终由你来确认。



第13页 · 费用与成本


很多人担心费用，我来算一下：


- Notion：免费或 $8/月

- LLM API：按用量，个人用户每月大概 $5-20

- Agent 框架：OpenClaw 开源免费，Dify 有免费套餐


总体来说，每月 $10-30 就能跑一个不错的 Notion AI Agent。


比雇一个助理便宜 100 倍。

我们现在处于 AI Agent 的早期阶段。


2023年，大家开始用 ChatGPT 这样的 AI 工具。

2024年，AI Agent 开始兴起，能自主完成任务。

2025年，多个 Agent 协作，处理复杂工作流。

未来，Agent 会成为每个人工作的标配。


你现在学的，是未来的基础技能。
